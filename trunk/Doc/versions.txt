Versions:

Json.NET comes in different versions for the various .NET frameworks.

-DotNet:
  .NET latest (4.0)

-DotNet35:
  .NET 3.5, Mono

-DotNet20:
  .NET 2.0

-Silverlight:
  Silverlight 3.0

-Phone:
  Windows Phone 7

Microsoft stopped supporting the Compact Framework in Visual Studio 2010.
For a Compact Framework 3.5 build download Json.NET 3.5.
